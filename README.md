# ODDSGOD

A cyberpunk prediction market aggregator.

## Features
- **Live Wire Header**: Real-time volume and trade ticker.
- **Slot Machine Feed**: Heavy, tactile feed with spring physics.
- **Anomaly Detection**: Whale alerts and Arbitrage opportunities.
- **Versus Mode**: Visual odds comparison.

## Tech Stack
- Next.js 14+ (App Router)
- Tailwind CSS 4
- Framer Motion
- Zustand

## Getting Started

```bash
npm install
npm run dev
```
